# job4j_forum

### Описание проекта 
Классический форум развернутый на Heroku. Простая функциональность добавление обсуждаемой темы, комментировать, удалять и исправлять!

### Использованные технологии

<ul>
  <li>Spring Boot</li>
  <li>Liquibase</li>
  <li>PostgresSQL</li>
  <li>Heroku</li>
</ul>
